# 🤝 Contributing to Aktivost

Thank you for considering contributing to **Aktivost** — an anonymous media platform for activists, journalists, and whistleblowers operating in high-risk environments.

---

## 🛡️ What Is Aktivost?

Aktivost allows users to upload sensitive media via Tor with full metadata removal, encryption, and optional public display through a clearnet Progressive Web App.

This project is focused on **privacy, decentralization, and resistance to censorship**. Every line of code can make a real difference to someone who needs to speak out — safely.

---

## 🚀 MVP Areas Needing Help

| Area | Description |
|------|-------------|
| Flask Blueprints | Upload logic, Redis queues, moderation routes |
| Plugin System | Hot-loading, signed ZIPs, PGP verification |
| Tailwind Templates | Responsive admin/mod dashboards |
| Redis/Queue Workers | Metadata scrubber, scanners, archive jobs |
| Security | PoW, anti-fingerprinting, metadata defense |
| PWA Viewer | Clearnet .org viewer using Tailwind + JS-lite |

---

## 📦 Plugin Ideas You Can Build

- `voice_mod`: Audio distortion plugin
- `video_live_plugin`: Anonymous livestream + chat
- `wallet_giftcard_plugin`: Gift card to Monero donation handler
- `clamav_scanner`: Virus/malware plugin
- `cold_storage_trigger`: Archive older files automatically

## ✅ How to Get Started

1. Fork the repo
2. Clone your fork and set up the environment (`README.md` will guide you)
3. Pick a `good first issue` or suggest a new one
4. Follow the coding and security guidelines in `/docs/`
5. Submit a PR!
